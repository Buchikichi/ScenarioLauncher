「ぶちきちの大冒険」の作り方 - マップを表示する

マップ画像

二次元のマップ上を動き回るタイプのゲームは、画面をスクロールさせる事が当たり前のようになっています。
「ぶちきちの大冒険」では、ブラウザー内でスクロールしているように見えるのですが、大きな画像の一部分を表示して、表示開始位置を少しずつずらす事でスクロールのように見せています。

プログラムソース(JavaScript, スタイルシート)

$(document).ready(function() {
	var div = $('#map');

	div.prop('mX', 0);
	div.prop('mY', 0);
	scroll();
});
function scroll() {
	var div = $('#map');
	var mX = div.prop('mX');
	var mY = div.prop('mY');
	var position = -mX + 'px ' + -mY + 'px';

	div.css('background-position', position);
	mX += 8;
	mY += 1;
	div.prop('mX', mX);
	div.prop('mY', mY);
	setTimeout(scroll, 100);
}
#map {
	width: 320px;
	height: 240px;
	background-image: url("img/map100bg.png");
}
